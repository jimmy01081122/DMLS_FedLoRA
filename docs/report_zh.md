# 聯邦學習下 LoRA 變體方法在資源受限環境中的比較研究報告

## 摘要
本研究針對資源受限環境（單張 RTX 3050 GPU），探討了三種基於 LoRA 的聯邦微調方法：Standard LoRA、FFA-LoRA 與 RoLoRA。實驗採用 Qwen2.5-1.5B 模型對 AG News 文本分類任務進行 Non-IID 數據分佈下的訓練。結果顯示，Standard LoRA 在各類分佈下表現最穩定，而 FFA-LoRA 與 RoLoRA 展現了顯著的通訊成本節省（高達 62.8%）且成功消除了聚合偏差（Aggregation Bias）。

## 1. 研究背景與目的
隨著大型語言模型（LLM）的普及，如何在保護隱私且硬體資源受限的前提下進行有效微調成為重要課題。聯邦學習（Federated Learning）結合參數高效微調（PEFT）技術如 LoRA，為此提供了可行方案。本研究旨在比較不同 LoRA 變體在通訊效率、聚合穩定性及 Non-IID 魯棒性方面的表現。

## 2. 實驗方法
### 2.1 模型與數據集
- **基礎模型**：Qwen2.5-1.5B (4-bit NF4 量化)
- **數據集**：AG News (四分類文本任務)
- **Non-IID 模擬**：使用 Dirichlet 分佈 ($\alpha=0.1, 0.5, 10.0$) 分配數據至 5 個客戶端。

### 2.2 比較方法
- **Standard LoRA**：同時訓練並傳輸 $A$ 與 $B$ 矩陣及分類頭。
- **FFA-LoRA**：凍結 $A$，僅訓練並傳輸 $B$ 及分類頭。
- **RoLoRA**：交替訓練 $A$ 或 $B$，每輪僅傳輸更新的部分及分類頭。

## 3. 實驗結果分析

### 3.1 通訊成本與準確率權衡 (Alpha=10.0)
| 方法 | 最終準確率 | 最終損失 (Loss) | 總通訊量 (MB) | 節省比例 |
| :--- | :--- | :--- | :--- | :--- |
| Standard LoRA | 0.8753 | 0.3622 | 250.78 | 0% |
| FFA-LoRA | 0.8380 | 0.4637 | 93.28 | 62.8% |
| RoLoRA | 0.8539 | 0.4192 | 119.53 | 52.3% |

### 3.2 Non-IID 程度的影響
| Alpha | Standard Acc | FFA-LoRA Acc | RoLoRA Acc |
| :--- | :--- | :--- | :--- |
| 10.0 (IID) | 0.8753 | 0.8380 | 0.8539 |
| 0.5 (Mid) | 0.6725 | 0.6211 | 0.6303 |
| 0.1 (Extreme) | 0.4155 | 0.3220 | 0.3130 |

實驗觀察到，隨著 $\alpha$ 減小（數據異質性增加），所有方法的性能均顯著下降，但 Standard LoRA 憑藉較大的參數容量展現了更好的魯棒性。

### 3.3 Aggregation Bias 分析
*   **現象說明**：在實驗初期（Round 1-2）所有方法的 Bias 均趨近於 0。
*   **深度分析**：Standard LoRA 在後期（Alpha=0.1, Round 5）出現了約 0.0089 的相對聚合偏差，這是由於 $A$ 與 $B$ 同時更新產生的非線性效應。
*   **結論**：FFA-LoRA 與 RoLoRA 由於每輪僅更新一個矩陣，在數學上保證了聚合的線性，成功將 Aggregation Bias 維持在 0，提高了聯邦學習的理論穩定性。

## 4. 數據規模影響 (Quick vs. Balanced)
對比每輪 10 樣本與 300 樣本的實驗：
*   **極低數據量**：模型無法收斂，Acc 維持隨機水平 (25%)，且數值不穩定。
*   **適中數據量**：300 樣本足以驅動 LoRA 權重向正確方向移動。對於 1.5B 模型，數據量必須跨過臨界閾值才能產生有效學習。

## 5. 研究結論
1. **通訊效率**：FFA-LoRA 是極度頻寬受限環境下的最佳選擇，節省了 62.8% 的開銷。
2. **性能平衡**：RoLoRA 在性能上更接近 Standard LoRA，同時節省了約一半的通訊量。
3. **穩定性**：FFA 與 RoLoRA 成功消除了聚合偏差，這對長期聯邦訓練的穩定性至關重要。
4. **硬體驗證**：RTX 3050 (6GB) 配合 4-bit 量化完全能支撐大型語言模型的聯邦學習研究。
